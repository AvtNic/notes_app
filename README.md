# My Notes.
Минималистичный электронный блокнот для быстрого создания, структурирования и безопасного хранения ваших мыслей. 
Проект разработан на Python с фокусом на интуитивный интерфейс и высокую скорость работы, чтобы ни одна важная идея не была утеряна.
Веб-приложение на Python, реализует полноценный сервис управления текстовыми заметками (CRUD). 
Проект демонстрирует навыки построения надежной архитектуры, работы с базами данных.

## Возможности
- Создать, посмотреть, изменить, удалить заметку
- Сохранение в SQLite

## Установка
bash
git clone https://github.com/USERNAME/notes_app.git
cd notes_app
pip install -r requirements.txt
python app.py
Открыть в браузере: http://127.0.0.1:5000

##  Технологии
•	Python 3.11
•	Flask 3.0
•	SQLite
