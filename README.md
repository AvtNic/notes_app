# notes_app
My Notes — Python web application that implements a full-fledged text note management service (CRUD)

Мои Заметки — минималистичный электронный блокнот для быстрого создания, структурирования и безопасного хранения ваших мыслей. 
Проект разработан на Python с фокусом на интуитивный интерфейс и высокую скорость работы, чтобы ни одна важная идея не была утеряна.
Это ваш персональный цифровой блокнот на каждый день. Фиксируйте идеи, организуйте списки дел и наводите порядок в мыслях в пару кликов.


Веб-приложение на Python, реализующее полноценный сервис управления текстовыми заметками (CRUD). 
Проект демонстрирует навыки построения надежной архитектуры, работы с базами данных, аутентификации пользователей и организации удобного REST API.
Проект выполнен с использованием базы данных SQLlite и фреймворка Flask 3.0.
Приложение позволяет создавать редактировать удалять заметки с указанием текущей даты. К просмотру доступен весь список заметок.

## Установка
```bash
git clone https://github.com/AvtNic/notes_app.git
cd notes_app
pip install -r requirements.txt
python app.py
Открыть в браузере: http://127.0.0.1:5000
Технологии
•	Python 3.11
•	Flask 3.0
•	SQLite
